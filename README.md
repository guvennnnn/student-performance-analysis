# Student Performance Analysis

Bu projede öğrencilerin demografik, akademik ve sosyal özellikleri kullanılarak
öğrencilerin final notlarının tahmin edilmesi amaçlanmıştır.

## Veri Seti
Çalışmada Kaggle üzerinde paylaşılan Student Performance veri seti kullanılmıştır.

## Yöntem
- Exploratory Data Analysis (EDA)
- Veri ön işleme (encoding, scaling)
- Linear Regression
- Random Forest Regression

## Değerlendirme
Modeller MAE, RMSE ve R² metrikleri kullanılarak karşılaştırılmıştır.

## Kullanılan Araçlar
- Python
- Jupyter Notebook
- Pandas
- Scikit-learn
